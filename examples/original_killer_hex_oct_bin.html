An original "computer killer" sudoku

Rules: normal killer sudoku rules apply, with the exception that cages totals are not written with standard decimal numbers (base 10). Instead, each cage total is either: a binary number (base 2), an octal numbers (base 8), or an hexadecimal number (base 16).

Use number keys to enter digits, Backspace to erase, hold Ctrl for center marks and Shift for corner marks.
